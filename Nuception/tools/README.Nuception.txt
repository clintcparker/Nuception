
Nuception

A package to build NuGet packages.
This package was designed specifally to address packages built 
using the TFS - Nugetter integration.

Please update your nuspec file with the appropriate values. 

To avoid conflicts, the the nuspec file has been configured 
to use the nuget_tools and nuget_content directories instead of the 
default tools and content directories.

http://www.codeproject.com/Articles/209522/PowerShell-Script-Examples-for-NuGet-Packages
